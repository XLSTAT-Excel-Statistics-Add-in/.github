## 🧭 What It Is
XLSTAT for Windows is a statistical software package delivered as an add-in for Microsoft Excel. Instead of exporting your numbers to a separate program, you keep them in Excel and let XLSTAT add a tab full of methods to the ribbon: descriptive statistics, regression, principal component analysis (PCA), a wide range of tests, and data visualization. If you have ever wondered what XLSTAT is, the short answer is that it is Excel with a professional statistics toolkit bolted on, working on the very cells you are already using.

## ⭐ Key Features
> All of these tools live on a single XLSTAT tab that appears in Excel after installation, so your analysis never leaves the spreadsheet.

- 📈 **Regression and modeling** — Fit linear, logistic, and other models to explain relationships and forecast outcomes.
- 🔬 **Statistical tests** — Run parametric and non-parametric tests, from t-tests to ANOVA and beyond.
- 🧮 **Descriptive statistics** — Summarize, cross-tabulate, and profile your datasets in a few clicks.
- 🧭 **PCA and multivariate methods** — Explore structure and reduce dimensionality in complex data with PCA and related techniques.
- 🎨 **Data visualization** — Produce charts, biplots, and diagnostic graphics that sit alongside your Excel data.

## ⚙️ Setup
1. Click the **Download** button above to get the latest version of XLSTAT.
2. Run the downloaded installer and follow the on-screen setup steps.
3. Open Microsoft Excel, confirm the XLSTAT tab is on the ribbon, and begin your free trial or activate a license.

## 👣 First Steps
1. Start Microsoft Excel after installing XLSTAT and find the XLSTAT tab on the ribbon.
2. If you don't see it, enable the add-in through Excel's Options under Add-ins and restart Excel.
3. Highlight the cells that contain your data.
4. Pick a method from the XLSTAT tab, adjust the options in the dialog, and run it.
5. Read the tables and charts XLSTAT places back into your workbook.

## ❓ Questions
**Is XLSTAT free?**
XLSTAT is a paid add-in offered with a free trial. The trial lets you test the tools inside Excel for evaluation; using it beyond the trial period requires a purchased license.

**Which versions of Windows are supported?**
XLSTAT is built for modern 64-bit Windows, including Windows 10 and Windows 11, together with a supported edition of Microsoft Excel for Windows.

**Is my data secure?**
Your analysis runs on the local Excel workbooks you open, so your data stays under your control. Keep regular backups and update Excel and the add-in as new releases arrive.
